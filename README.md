🏗️ Cadastro de Imóveis — Flask

Aplicação web fullstack desenvolvida em Flask + SQLite, com autenticação de usuários, cadastro de imóveis e formulário de comunicação de obras.

O sistema foi pensado para ser leve, responsivo e de fácil uso em campo, especialmente em canteiros de obra e ambientes corporativos.

Interface construída com Bootstrap 5, priorizando experiência mobile-first.

🚧 Status: Projeto em desenvolvimento.

✨ Funcionalidades

🔐 Autenticação de usuários (login e controle de sessão)

🏠 Cadastro e gerenciamento de imóveis

📝 Formulário de comunicação de obras

📊 Interface responsiva (Bootstrap 5), adaptada para celular e desktop

💾 Banco de dados SQLite, simples e portátil

🚀 Tecnologias Utilizadas

Banco de Dados: SQLite

Frontend: HTML, CSS, JavaScript, Bootstrap 5

Autenticação: Flask-Login / Sessions

📦 Como executar localmente
# Clone este repositório
git clone https://github.com/seu-usuario/cadastro-imoveis-flask.git

# Acesse a pasta do projeto
cd cadastro-imoveis-flask

# Crie e ative um ambiente virtual
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows

# Instale as dependências
pip install -r requirements.txt

# Execute a aplicação
flask run


Acesse em: http://localhost:5000
